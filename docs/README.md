# Outflō Documentation

This directory contains the permanent documentation for the Outflō codebase.

## Structure

docs/

  conventions/
    Repository rules and standards.

  architecture/
    System design and technical contracts.

## Conventions

Conventions define repository laws and are not versioned.

Examples:
- naming-canon.md
- code-style-locked.md

## Architecture

Architecture documents describe system designs and may evolve over time.

Architecture documents are versioned.

Example:

environment-engine/
  v1.md
